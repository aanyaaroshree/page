#Overview

This project is a standalone webcam dashboard built using HTML, CSS, and Vanilla JavaScript. The application provides webcam control functionality along with a real-time activity log and digital clock.

#Features

- Webcam video display
- Start camera functionality
- Stop camera functionality
- Capture simulation with flash effect
- Real-time activity log
- Live digital clock
- Dynamic button state highlighting
- Responsive dashboard layout
- Modern user interface design

#Technologies Used

- HTML
- CSS
- JavaScript 
- Browser APIs
 - MediaDevices API
 - DOM API
 - Event API
 - Timer API

#Running the Project

 1. Download or clone the repository.
 2. Open index.html in any modern web browser.
 3. Allow camera permissions when prompted.
 4. Use the Start, Stop, and Capture controls.
